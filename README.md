#frame 框架简介
   结合springMVC + spring + mybaites + netty + (springColud可以根据项目需求再定是否使用 springColud服务端我用的是nacos) 用idea搭建的技术,主要是    基于gui 自动生成来实现的，基础的增删改查可以直接导入自动生成的就可以实现，不用自己去写相应的代码。自定义的就需要自己去书写了。大大提高了效率和节省时    间【同时这个框架实现了netty tcp 开发】
